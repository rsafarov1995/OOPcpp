# OOPcpp
Задание №1.

Спроектировать и реализовать шаблонный класс Container со следующими возможностями:

    - добавить новый элемент в начало / конец;

    - удалить первый / последний элемент;

    - получить значение первого / последнего элемента;

    - перебрать все элементы (получить в цикле все элементы по ссылке);

    - узнать кол-во элементов / проверить на пустоту;

    - поменять содержимое с другим экземпляром контейнера;

    - переставить все элементы в обратном порядке;

    - очистить контейнер.

Класс должен быть эффективным и универсальным 
(должен подходить для хранения экземпляров любых типов и использования где угодно). 
Каждый метод, кроме двух последних, должен иметь сложность O(1).
